# Bulk Certificate Generator

A professional, automated tool built with Python and Streamlit to generate bulk certificates. Simply upload an Excel sheet of names and a base image template to create bulk certificates in seconds.
.
This is made purely using Gemini.
I have no code exprience.

Upload File Instructions:
Input Excel file added in repo. 
A4 dimension(if possible else any size will do) amd Jpeg Format for template of certificate.
